# Hire Hydrant Monitoring system
Web application 


**Pressure, Water level sensor** ---Modbus---> **IoT gateway** ---MQTT---> **On premises server(DB & Alert)** ---HTTP---> **Web UI**
<br>

Prerequisite:
* Install python
* Install node
* Install Postgres as service

Run
1. python datagen.py
2. node mqtt2postgres.js
